node 简单服务创建

发表于 2017-03-12   |   分类于 node服务   |     |    阅读次数

首先是安装node.js

点击这个链接进入nodejs首页 http://www.nodejs.org

验证是否安装成功

window+R 打开输入 cmd 再输入 node -v 查看版本号

#安装node之后安装express模块插件

npm install express

创建node server 如下

var http = require('http'); //请求http

创建服务器

//req 表示客户端请求
//res 服务器返回
http.createServer(function(req,res){
    res.end('create a server success!');
}).listen(3000); //创建一个服务器端口 端口号为3000

打开浏览器输入:http://localhost:3000

node server //server是我们所创建的文件

解决cmd下显示乱码问题

res.setHeader('content-type','text/html;charset=UTF-8');
